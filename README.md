# orbiter

思考メモツール「orbiter」——進行中の思考の中断状態を保存し、有限な注意を「いま開いている問い」に向け続けるための装置。第一段(循環の検証機)のプロトタイプ。

仕様・概念は `doc/` を参照:
- `doc/orbiter-spec-phase1.md` — 第一段の開発仕様
- `doc/思考メモ_ツール概念_最終形.md` — 思想と原理
- `doc/思考メモ_周回モデル.md` — 視覚・力学モデル
- `doc/phase1-decisions.md` — 実装時の判断メモ

## 起動

```bash
npm install
npm run dev   # http://localhost:5173
```

初回はまっさらな空に「例のデータで始める」が出る。右上の「···」からもいつでもサンプル・書き出し・読み込みができる。

## 使い方

UIの原則: **常設の道具を持たない。すべては召喚される。**

| 操作 | やり方 |
|---|---|
| 打ち上げ(書く) | **そのまま打ち始める**。入力行が浮かび、Enterで粒になる |
| 続ける(追記) | 粒をクリックして選択→打ち始める(幹が繋がり、親の時計が巻き戻る) |
| 付箋 | 粒を選択→入力中にTabで「続ける⇄付箋」切替、またはグリフ「付箋」。「?」で始めると開いた疑問(沈みにくい) |
| 閉幕 | 粒を惑星へドラッグして落とす / Delete / グリフ「閉幕」。即時退場、一言は不要 |
| 析出 | 粒を選択→グリフ「析出」で名を与えテーマにする |
| 合流 | Ctrl+クリックで複数選択→打ち始める |
| 訂正 | ダブルクリック、またはグリフ「訂正」(時計は動かない) |
| 検索・蘇生 | `/` または Ctrl+K。生死横断の全文検索。死んだ粒は一言(必須)を添えて蘇生 |
| テーマの尾 | テーマ名クリック、またはグリフ「尾」。右から帯が滑り込む。開いたまま打てば先端に追記 |
| ズーム | ホイール。遠=点と塊の星空(全天)、近=全文で読む(地表寄り) |
| 回転(ダイヤル) | 空をドラッグ。接触順に思考を撫でるパラパラ見返し |
| フォーカス | 粒をクリック=その粒に乗り、画面中心へ。星座(幹・付箋)が召喚される |
| Esc | 段階後退: 入力 → 検索/帯 → 選択解除 |

## 力学

- 時間は壁時計ではなく**滞在時間**で流れる。ウィンドウにフォーカスがある間だけ生態系時刻が進み、不在中は凍結
- 触られない粒は冷え(文字が小さく薄く、外周へ)、滞在3時間で警告なく漂流する。帯域(30粒)からあふれても最も冷えた粒から漂流する
- 時計を巻き戻すのは「手を通した粒の追加」のみ(付箋・追記・蘇生)。閲覧・選択は代謝ゼロ

調律パラメータはすべて `src/params.ts` に集約。データはlocalStorageに保存され、「書き出し」でJSONエクスポートできる。
