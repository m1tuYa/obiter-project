# 思考メモ — 周回モデル(「今」の面の設計)

*2026-07-15 時点。設計フェーズの成果。概念(ツール概念_最終形.md)の上に立つ、「今」の面の視覚・力学モデル。*

---

## 0. 位置づけと評価基準

- このモデルは思考の**絵**ではなく**循環器**。評価基準は「思考の形に似ているか」ではなく「誕生と死の循環が回るか」
- 循環の四条件: **①去れるか**(粒が視界から退場できる) **②戻る道があるか**(再会の経路) **③戻ったとき書けるか**(回帰が変換を通る) **④書くたびに回るか**(ポンプ)
- 反面教師の再診断: マインドマップは①で死んでいる(何も去れない。不在がなければ再会もない)。ツイッターは②が自分に向いていない+③が禁じられている(凍結)。ToDoは③がない(変換なき回帰=催促)
- **ポンプは「書くこと」のみ**。一粒書くたびに系が一歩回る(誕生・押し出し・沈降が同時に駆動される)。管理習慣をポンプにしたツールは続かない

## 1. 全体像

> **中心に、閉じた思考が堆積して育つ惑星(死者の国)。その周りを、生きている思考たちが軌道の高さ=温度で周回する。触られない粒は次第に高軌道へ遠ざかり、やがて重力圏を離れて漂流帯(雲)へ。閉じられた粒は大気圏に突入し、惑星の質量になる。雲からは時折、彗星が帰ってくる。**

- 開いたらせん(放射型)の発展形。閉じた輪ではない——厳密に同じ場所に戻るものはない(回帰は常に変化を伴う)
- 生=運動(回り続けること)、死=運動の停止。画面を開けば、自分の頭がまだ回している問いが文字通り回っている
- 中心=惑星に置かれるのは死ではなく**蓄積**。生の側(周回する粒)が構図の主役で、死は縁と中心で静かに起きる

## 2. 部品表(対応)

| 概念 | この宇宙での姿 |
|---|---|
| 断片 | 衛星(粒) |
| 手を通した接触 | 軌道維持の噴射。触らない粒は軌道が減衰する |
| 温度 | 軌道の高さ。熱いほど低軌道で速く大きく、保留ほど高軌道で遅く小さく |
| 沈降(デフォルトの死) | 漂流。重力圏を離れ、外の雲へ去る |
| 閉幕(明示の死) | 大気圏突入。印一つで落とす。一瞬燃えて、惑星に積もる |
| 死者の国 | 惑星。突入した粒で**育つ**。掘れば内部は地層の断面 |
| 地層・圧縮 | 惑星の内部構造。俯瞰では厚みとしてのみ見える |
| テーマ | **深さを持つ粒**。俯瞰では代表粒(最も熱い先端)一つ+点と線の尾。開いたときだけ内部(入れ子の生態系)が現れる |
| 幹(系譜) | 尾。代表粒の後ろに引かれる点と線 |
| 予定・締切 | **自筆の彗星**。書き手が軌道(いつ戻るか)を与えた粒。近日点で触れば捕獲、無視すればまた遠ざかる |
| 転生 | **軌道なき彗星**。漂流帯から、脳が引き戻したときだけ帰ってくる。予約できない |
| 蘇生 | 惑星からの引き上げ。死んだ粒を再び軌道に乗せる。**一言を要求する**(なぜ戻すのか)——系譜が死を跨いで継続する |
| まとめなおし(合流) | 俯瞰上で生き残りを選び、複数の幹を持つ新しい粒を書く。専用の場は不要 |

## 3. 力学の規則

### 誕生
- 書く=打ち上げ。打ち上げ位置の判断は存在しない(運動する系に定位置はない)
- **幹は、書いたときに乗っていた参照系が決める**: 惑星系(どの粒にも乗っていない)で書けば無所属の粒が湧き、あるテーマの粒に並走しながら書けばその先端に繋がる。宛先の判断はゼロ——書く前に居場所は決まっている

### 生存
- 軌道維持は「手を通した粒の追加」のみ。最小単位は付箋一枚。閲覧は代謝ゼロ(眺めるだけの延命は不可能)
- 種類ごとの優遇なし。観測される寿命の差は接触パターンの現れ

### 死
- 突入(閉幕)は印一つで軽い。惑星の成長という意味の反転により、捨てる=建設になる(溜め込み心理への最終回答)
- 漂流(沈降)は無音・無警告。「沈みかけ」を目立たせない——救済延命(空虚な接触)を誘発するため
- 突入した粒(閉じた問い)と漂流した粒(未決のまま冷めた問い)は**別の場所に眠る**。転生の種は主に漂流帯にある

### 帯域
- 生きて周回できる粒の総量は注意の帯域で規制される(環境収容力)。あふれれば最も冷えた粒から漂流する
- 生態系の時間は滞在時間で流れる。不在中は凍結

## 4. ズーム三景

- 縮退は半径固定ではなく**半径×倍率の関数**。ズームが「眺める」と「読む」を連続的につなぐ統一原理
- **最遠=全天**: 点と塊(テーマ)の星空。テーマ名だけが微かに浮かび、全体がゆっくり回る。眺める場所、圧縮の兆し(塊の肥大)を感じる場所。線は出ない
- **中間=空域**: 塊が語になり、尾が見える。棚卸しの場所
- **最近=地表**: 視点が惑星の地表に降りる。視野は180度以下——足元に自分の蓄積した過去(地面=死者の国)、頭上に最内周+数日分の熱い粒が全文で回る。書く・触る・日々のデフォルト。打ち上げは地面から発つ(射出台=地表)
- **ダイヤル(回転)=パラパラ見返し**: 角度は接触順なので、地表から空を見回す回転は、直近から順に思考を撫でる行為そのもの。再会の第一経路(時間的近接)の身体動作
- **視点は揮発**: ズームと回転の状態は保存しない。起動は常に地表から(配置と同じく、視点も解釈)
- **枠の廃止**: カード枠は保管ツールの遺物。暗い空に枠なしの文字が浮かぶ。粒のラベルは新設しない——テーマは析出時の名、無所属の粒は冒頭の数語(自分の言葉で書かれた粒は冒頭が既に要点)。ズームに応じて本文の見える量が伸縮する(数語→一行→全文)

## 5. 表示の規律

- **縮退の四段**: 熱い側は全文で読める / 中間は一〜二行 / 外周は数語または無文字の点 / テーマの従属粒は代表粒の尾(点と線)。中心視と周辺視の構造。段の適用はズームと半径の積で決まる(→4章)
- **全文の一望は死のデフォルト化の報酬**: 生者が帯域内に抑えられているからこそ、生きているものすべてを原文で一望できる(キーワード地図への切り詰め不要)
- **注意=参照系の選択**: デフォルト(惑星固定系)ではすべてが緩やかに周回する。特定の粒・方角に着目すると、その参照系に乗り移り、対象は静止して見え、残り(惑星を含む)が流れて見える。読む・書くための静止は、時間を止めずに参照系を合わせることで得る
- **運動は情報であって演出ではない**: 速度差=温度の表現。全体の角速度は地質学的に遅くてよい。周辺視が「まだ生きている」と感じる最低限
- **構造は召喚されるもの**: 線は常設しない。粒の選択時のみ、その粒の幹・付箋・細い幹が浮かぶ(保存された事実のみ。意味的類似の計算表示は当面なし)。選択は接触に数えない
- 惑星の成長は見えるが**祝われない**(数値・バッジなし)。ゲーム化との境界
- 比喩は力学(運動・軌道・減衰・突入)のみ輸入し、見た目は抽象に保つ(地球の絵・炎のエフェクト等の具象化は装飾への堕落)

## 6. 廃止された部品

- **作業台** — 俯瞰そのものが全生存粒を一望に置くため、圧縮の専用室は冗長になった。配置を手で動かす運動は失うが、配置は解釈であり保存しないと決めた時点で本質ではなかった
- **机(二態のボード)・林(テーマの地図)・左ペインのストリーム** — 周回の俯瞰に統合
- 入力欄は俯瞰では非表示(書く行為は参照系+打ち上げとして常時可能)

## 7. 原則との照合(合法性の確認)

- 配置(軌道)は接触履歴から計算される。保存しない=「配置は解釈」の原則に適合
- 生かす操作は重く(言葉を要求)、殺す操作は軽い(印一つ)。蘇生は一言を要求=生の側
- 時間駆動の自動再提示(間隔反復)は禁忌のまま。**彗星が合法なのは軌道を自分で書いたときのみ**——禁忌の線は「時間駆動か」ではなく「誰が軌道を書いたか」
- 疑問のリスト化はしない。開いた付箋は低軌道に浮かび続けることで追跡なしに保たれる

## 8. 自覚されたリスク

- **閉幕の快感の逆流**: 惑星を育てたくて、まだ開いているべき問いを早閉じする誘惑。成長を祝わない抑制で対処
- **具象化の誘惑**: 木のモデルが陥りかけた罠。力学だけを借りる規律
- **内周の容量**: 全文で読めるのは一視野あたり3〜4粒が正直な限界。ズーム三景の導入によりほぼ解消(読みたければ寄る)。地表視点の初期視野に何粒置くかの調整のみ残る

## 9. 周辺の大枠(案・詳細未確定)

- **テーマ内部**: 開く=画面遷移ではなく、代表粒の参照系に乗り、尾がズームして読める帯になる。着地点は冒頭ではなく**先端**(最新の生き残り+浮いている付箋=中断状態)。遡行は尾を上流へ歩く。圧縮で生まれた粒は**結び目**で、開けば親の支流に降りられる——深さ=結び目の段数=地層
- **惑星の断面**: 縦軸は堆積順(突入した順)の考古学。世代(圧縮回数)の軸はテーマの結び目が担うため持たない。死者は二つの国に分かれる——惑星=閉じた問い(閉幕の一言が墓碑銘)、雲=未決のまま冷めた問い。**掘るのは惑星のみ、雲は眺めない**(向こうから帰るか、検索で呼ぶ)。参照(読むだけ、代謝ゼロ)と蘇生(一言を要求)は別の操作
- **スマホ=射出台**: 開けばカーソルだけ。書いて閉じる。俯瞰なし。粒はすべて無所属で打ち上げ(出先でのテーマ選択は分類判断の復活なので置かない)、帰還後に幹を繋ぐ(繋ぐ行為が接触になる)。不在中の空は凍結し、打ち上げた粒は最内周に積もって帰還を待つ
